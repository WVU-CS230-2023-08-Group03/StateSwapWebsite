Listing Folder
    This folder contains code for creating a list and the corresponding listing.

createListing.jsx
    The createListing.jsx file in this folder contains the JavaScript XML code for creating a listing on our store page. This code gives the user the ability to upload a picture, a title, and a breif description about their product that they are offering on the store page.

createListing.css 
    The style for the createListing.jsx file.

Listing.jsx 
    The Listing.jsx file contains the listing generated using created listing from createListing.jsx. This file showcases the content of the listing in a readable format.

Listing.css 
    The style for the Listing.jsx file.
