Report Folder
    Contains the code for the Report buttons beneath the Listings and the page with all of the submitted reports

Report.jsx
    File with JavaScript XML code that allows the user to click a report button on a Listing and submit a report with drop down category, breif description, and a title that sums up the description

Report.css  
    The style for the Report.jsx

reportView.jsx  
    File with JavaScript XML code that displays a styled page with all of the previously submitted reports in a sequential order for an admin to view and to help them monitor the website.
