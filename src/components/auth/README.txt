SignIn.jsx
    Description: Grabs the email and the password from the sign in field and sends the information to Firebase, and checks to see if it's in the database. 

SignOut.jsx
    Description: This signs the user out of their account. 

SignUp.jsx
    Description: The user can create an account with the aid of this code. Once the user puts in their email and desired password into the text fields, it will be sent to the backend.
    Once the data is sent to the backend, an entry is created for the user with a hashed version of their password, and their email. To send this data to the backend, the user will need to click the button to create their profile.
